"Jenkins - Step by Step", 2022

Mentor: Lev Gorfel | 2022

This tool allows us to build new code versions, run unit tests, run ui and api system tests, and much more.

Jenkins is a popular open source tool for continuous integration and build automation. Jenkins allows you to follow a predefined list of steps and is used for example to compile java source code and build a jar from the resulting classes. The trigger for this execution can be based on time or event. For example, you can compile a java based application every 20 minutes or after a new commit in the corresponding git repository.

Jenkins monitors the execution of steps and allows you to stop the process if one of the steps fails. Jenkins can also send notifications when a build succeeds or fails. jenkins can be extended with additional plugins. For example, you can install plugins to support the creation and testing of android applications.

As part of the course, you will learn how to work with docker (another extremely useful tool), install jenkins on a local unix server, get acquainted with the graphical shell of the system, learn how to integrate jenkins with java projects and projects from the githib repository. We will run unit tests together first, and later we will set up and run entire projects using selenium webdriver and postman.
